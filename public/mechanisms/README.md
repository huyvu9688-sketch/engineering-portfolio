# Mechanism illustrations

The motor-sizing calculator (`/tools/motor-sizing`) shows one of these
images on each mechanism card. Save the line-art illustrations here with
**exactly** these filenames (PNG). Until a file is present, the card
automatically falls back to a simple built-in line icon — so the app
works with or without them.

| File | Mechanism |
| --- | --- |
| `ball-screw.png`    | Ball / lead screw |
| `belt-conveyor.png` | Belt conveyor |
| `rack-pinion.png`   | Rack & pinion |
| `index-table.png`   | Rotary index table |
| `direct-drive.png`  | Direct drive (motor rotor) |

Tips: square-ish framing, white or transparent background, ~512–1024 px.
They render at ~96 px tall with `object-contain`, so any aspect ratio is
fine.
